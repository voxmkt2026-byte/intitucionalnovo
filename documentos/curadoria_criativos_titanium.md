## Regras de Compliance para Criativos

> [!CAUTION]
> **Criativos para serviços financeiros são os mais reprovados no Meta.** Seguir estas regras evita bloqueio da conta de anúncios.

### ❌ NÃO usar nos criativos
| Proibido | Motivo |
|---|---|
| "Aprovação garantida" | Garantia de resultado financeiro |
| "Contemplado em X meses" | Promessa de prazo de contemplação |
| "Sem juros" (isolado) | Pode ser interpretado como enganoso |
| "Renda extra" / "Investimento" | Categoria restrita no Meta |
| "Crédito aprovado pra todos" | Promessa de aprovação universal |
| Fotos de dinheiro/notas | Trigger de reprovação automática |
| Antes/Depois financeiro | Categoria restrita |

### ✅ PODE usar
| Permitido | Exemplo |
|---|---|
| Comparativo de parcelas | "R$ 990 vs R$ 2.800" |
| "Sem juros abusivos de financiamento" | Qualificado, não absoluto |
| "Parcela que cabe no bolso" | Benefício genérico |
| "Fale com um consultor" | CTA consultivo |
| "Aprovação facilitada" | Sem garantia |
| "Consórcio regulamentado pelo Banco Central" | Verdadeiro, referindo-se ao sistema |
| Depoimentos reais com autorização | Prova social legítima |

## Configuração de Tracking

### Placeholders no código (já inseridos):
- **GTM:** Substituir `GTM-XXXXXXX` pelo ID real do container
- **Meta Pixel:** Substituir `XXXXXXXXXX` pelo Pixel ID real
- **Evento de conversão:** `Lead` dispara no clique do WhatsApp automaticamente
- **Rastreamento de LP:** Cada mensagem WhatsApp inclui `[lp_nome]` para identificar origem

### Eventos configurados:
| Evento | Trigger | Destino |
|---|---|---|
| `PageView` | Carregamento da página | Meta Pixel |
| `Lead` | Clique no botão WhatsApp | Meta Pixel + GTM dataLayer |
| `whatsapp_click` | Clique no botão WhatsApp | GTM dataLayer |
| `lp_source` | Texto da mensagem WhatsApp | Identificação da LP no atendimento |
